# BEYOND THE SKILL - Project Overview

## 🎯 Project Summary

A premium, cinematic landing page for "Beyond The Skill" - an elite gymnastics refinement brand owned by Rifda Irfanaluthfi, Indonesia's first and only Olympic gymnast.

## 🎨 Design Philosophy

**NOT a kids gym. NOT a cheerful academy.**

This is a luxury athletic performance brand with:
- Olympic documentary aesthetic
- Nike athlete campaign feel
- Minimalist brutalism
- Dark luxury UI with red accents
- Elite positioning for advanced athletes

## 📊 Technical Stack

```
Framework: Next.js 14 (App Router)
Styling: Tailwind CSS
Animation: Framer Motion
Fonts: Bebas Neue (display) + Inter (body)
Deployment: Vercel
Domain: Domainesia
```

## 🎨 Brand Colors

- **Black**: `#000000` - Primary background
